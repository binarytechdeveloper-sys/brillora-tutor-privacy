1. Introduction

Brillora Tutor is an educational application designed for teachers and tutors to manage classrooms, students, courses, tests, announcements, and live classes. By using Brillora Tutor, you agree to this Privacy Policy and our data handling practices.

2. Information We Collect

A) Tutor Personal Information

  • Full name
  • Email address
  • Phone number
  • Profile photo (optional)
  • Firebase Authentication User ID

B) Classroom & Student Management Data

  • Classroom details created by tutors
  • Student names and academic details (added by tutors)
  • Courses, test papers, questions, and announcements
  • Manual evaluation inputs and grading data

C) Communication Data

  • Chat messages between tutor and students
  • System notifications related to classes and activities

D) Location Data

  • Approximate location (optional) for tutor discovery and matching

E) App Activity & Diagnostics

  • Crash reports and error logs
  • App usage and performance analytics
  • Device or app identifiers (non-personal)

F) Payment Information

Payments (if enabled) are processed via trusted third-party services such as UPI apps or Razorpay. Brillora Tutor does NOT store bank details, card numbers, UPI PINs, or passwords.

3. How We Use Data

  • To create and manage tutor accounts
  • To manage classrooms, students, and courses
  • To conduct tests, evaluations, and announcements
  • To enable tutor-student communication
  • To send important notifications
  • To improve app stability, security, and user experience

4. Data Sharing & Third Parties

We do not sell or rent personal information. Data is shared only with essential service providers:

  • Firebase (Authentication, Firestore, Notifications)
  • Payment gateways when tutors initiate transactions

5. Data Security

All data is encrypted in transit using secure protocols. Firebase securely stores app data. While we follow best security practices, no system can guarantee absolute security.
