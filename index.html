1. Introduction

Brillora Tutor is an educational application designed for teachers and tutors to manage classrooms, students, courses, tests, announcements, and communication. By using Brillora Tutor, you agree to this Privacy Policy and our data handling practices.

2. Information We Collect

A) Tutor Personal Information

  • Full name
  • Email address
  • Phone number
  • Profile photo (optional)
  • Firebase Authentication User ID

B) Classroom & Student Management Data

  • Classroom details created by tutors
  • Student names and academic details (added by tutors)
  • Courses, test papers, questions, and announcements
  • Manual evaluation inputs and grading data

C) Communication Data

  • Chat messages between tutors and students
  • System notifications related to classes and activities

D) Location Data

  • Approximate location (optional) for tutor discovery and matching

E) App Activity & Diagnostics

  • Crash reports and error logs
  • App usage and performance analytics
  • Device or app identifiers (non-personal)

F) Payment & Subscription Information

If enabled, digital services such as subscriptions or visibility features are processed securely using Google Play Billing. Brillora Tutor does not store credit card numbers, bank details, or payment credentials.

3. How We Use Data

  • To create and manage tutor accounts
  • To manage classrooms, students, and courses
  • To conduct tests, evaluations, and announcements
  • To enable tutor-student communication
  • To process subscriptions and app features
  • To improve app stability, security, and user experience

4. Data Sharing & Third Parties

We do not sell or rent personal information. Data is shared only with essential service providers, including:

  • Google Firebase (Authentication, Firestore, Notifications, Storage)
  • Google Play Billing for subscription processing

5. Data Security

All data is encrypted in transit using secure HTTPS/TLS protocols. Firebase securely stores application data. While we follow industry best practices, no system can guarantee absolute security.

A) Delete Specific Data (Without Deleting Account)

Users can delete certain data, such as chat messages, directly within the Brillora Tutor app without deleting their account. Deleted chat data is permanently removed or anonymized from our systems.
